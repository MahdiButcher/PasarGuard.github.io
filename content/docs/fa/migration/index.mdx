به بخش راهنماهای مهاجرت خوش آمدید. اینجا راهنماهای مختلفی پیدا می‌کنید که بهتون کمک می‌کنه تنظیمات موجودتون رو منتقل کنید.

پاسارگارد یه راه‌حل یکپارچه برای دور زدن سانسور با رابط کاربری گرافیکیه - یه فورک قدرتمند از Marzban با امکانات پیشرفته و بهبودهای جدید.
