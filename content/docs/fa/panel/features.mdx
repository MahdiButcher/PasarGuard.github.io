## نمای کلی داشبورد

پنل PasarGuard یک رابط وب مدرن برای مدیریت زیرساخت پروکسی شما فراهم می‌کند. به داشبورد در `https://YOUR_DOMAIN:8000/dashboard/` دسترسی پیدا کنید

## مدیریت کاربران

### ایجاد کاربران

ایجاد و مدیریت کاربران پروکسی از طریق رابط بصری:

- **پروتکل تکی**: ایجاد کاربران با یک پروتکل (VMess، VLESS، Trojan، Shadowsocks)
- **چند پروتکل**: اختصاص چندین پروتکل به یک کاربر
- **محدودیت ترافیک**: تعیین محدودیت انتقال داده برای هر کاربر
- **تاریخ انقضا**: پیکربندی تاریخ انقضای اشتراک

### وضعیت کاربر

نظارت و کنترل حساب‌های کاربری:

- **فعال/غیرفعال**: فعال یا غیرفعال کردن فوری کاربران
- **استفاده از ترافیک**: نظارت بر مصرف داده در زمان واقعی
- **وضعیت آنلاین**: مشاهده کاربرانی که در حال حاضر متصل هستند
- **بازنشانی ترافیک**: بازنشانی شمارنده ترافیک کاربر در صورت نیاز

## مدیریت اشتراک

### لینک‌های اشتراک

PasarGuard لینک‌های اشتراک سازگار با کلاینت‌های محبوب تولید می‌کند:

- **V2Ray**: فرمت استاندارد اشتراک V2Ray
- **Clash**: فرمت پیکربندی Clash
- **ClashMeta**: فرمت پیشرفته Clash با ویژگی‌های اضافی

### کدهای QR

کدهای QR تولید شده خودکار برای پیکربندی آسان کلاینت:

- اشتراک‌گذاری مستقیم کدهای QR با کاربران
- راه‌اندازی سریع در دستگاه‌های موبایل
- پشتیبانی از تمام پروتکل‌ها

## مدیریت نود

### معماری چند نودی

توزیع زیرساخت خود در چندین سرور:

- **توزیع بار**: توازن ترافیک در نودها
- **توزیع جغرافیایی**: قرار دادن نودها در مکان‌های مختلف
- **افزونگی**: نودهای پشتیبان برای دسترسی بالا

### نظارت بر نود

پیگیری زیرساخت نود خود:

- **وضعیت سلامت**: نظارت بر اتصال و وضعیت نود
- **استفاده از منابع**: نظارت بر CPU، حافظه و پهنای باند
- **آمار ترافیک**: تحلیل ترافیک به ازای هر نود

## نظارت بر سیستم

### آمار ترافیک

نظارت جامع بر ترافیک:

- **آمار لحظه‌ای**: نمایش استفاده از ترافیک زنده
- **داده‌های تاریخی**: تاریخچه و روندهای ترافیک
- **تحلیل به ازای کاربر**: مصرف فردی کاربر
- **نمای کلی سیستم**: استفاده کل از پهنای باند

### معیارهای عملکرد

نظارت بر عملکرد سیستم:

- **استفاده از CPU**: بهره‌برداری از CPU سرور
- **استفاده از حافظه**: مصرف RAM
- **اتصالات فعال**: تعداد اتصالات همزمان
- **زمان فعالیت**: آمار در دسترس بودن سیستم

## ویژگی‌های خودکارسازی

### ربات تلگرام

یکپارچه‌سازی با تلگرام برای اعلان‌ها و مدیریت:

- **اعلان‌های کاربر**: هشدارهای خودکار انقضا و ترافیک
- **دستورات مدیر**: مدیریت پنل از طریق تلگرام
- **هشدارهای لحظه‌ای**: اعلان‌های وضعیت سیستم

### ابزارهای CLI

رابط خط فرمان برای خودکارسازی:

```bash
# مدیریت کاربر
pasarguard cli user create
pasarguard cli user list
pasarguard cli user delete

# مدیریت ادمین
pasarguard cli admin create --sudo
pasarguard cli admin list

# عملیات سیستم
pasarguard status
pasarguard restart
pasarguard update
```

## ویژگی‌های امنیتی

### پشتیبانی از TLS

اتصالات امن با TLS:

- **گواهی‌های خودکار**: مدیریت داخلی گواهی
- **گواهی‌های سفارشی**: استفاده از گواهی‌های SSL خود
- **پیکربندی SNI**: پشتیبانی از Server Name Indication

### پروتکل REALITY

مقاومت پیشرفته در برابر سانسور:

- **استتار ترافیک**: مخفی کردن ترافیک پروکسی
- **حریم خصوصی پیشرفته**: ویژگی‌های امنیتی بهبود یافته
- **مقاومت در برابر اثر انگشت**: اجتناب از شناسایی

## گزینه‌های پیکربندی

### تنظیمات ورودی

سفارشی‌سازی اتصالات ورودی:

- **پیکربندی پورت**: تعریف پورت‌های گوش دادن
- **تنظیمات پروتکل**: پیکربندی گزینه‌های خاص پروتکل
- **گزینه‌های امنیتی**: تنظیمات TLS و امنیت

### قوانین مسیریابی

مسیریابی پیشرفته ترافیک:

- **مسیریابی دامنه**: مسیریابی بر اساس نام دامنه
- **مسیریابی IP**: هدایت ترافیک بر اساس آدرس‌های IP
- **قوانین سفارشی**: ایجاد منطق مسیریابی پیچیده

## بهترین شیوه‌ها

### نگهداری منظم

- **پشتیبان‌گیری از پایگاه داده**: پشتیبان‌گیری منظم از پایگاه داده
- **به‌روزرسانی پنل**: نرم‌افزار را به‌روز نگه دارید
- **نظارت بر لاگ‌ها**: بررسی لاگ‌ها برای مشکلات
- **بررسی کاربران**: ممیزی دوره‌ای حساب‌های کاربری

### بهینه‌سازی عملکرد

- **برنامه‌ریزی منابع**: تخصیص منابع کافی
- **تنظیم پایگاه داده**: بهینه‌سازی عملکرد پایگاه داده
- **توزیع نود**: توازن بار در نودها
- **نظارت بر ترافیک**: پیگیری و تحلیل الگوهای استفاده
