## نمای کلی داشبورد

پنل PasarGuard یه رابط وب مدرن برای مدیریت زیرساخت پروکسی شما فراهم می‌کنه. برای دسترسی به داشبورد برید به `https://YOUR_DOMAIN:8000/dashboard/`

## مدیریت کاربران

### ساخت کاربر

کاربرای پروکسی رو از طریق رابط ساده بسازید و مدیریت کنید:

- **چند پروتکل**: چندین پروتکل رو به یک کاربر اختصاص بدید
- **محدودیت ترافیک**: حجم انتقال داده برای هر کاربر رو تنظیم کنید
- **تاریخ انقضا**: تاریخ اتمام اشتراک رو مشخص کنید

### وضعیت کاربر

اکانت‌های کاربری رو کنترل و مانیتور کنید:

- **فعال/غیرفعال**: کاربرا رو به صورت آنی فعال یا غیرفعال کنید
- **مصرف ترافیک**: استفاده از داده رو به صورت لحظه‌ای ببینید
- **وضعیت آنلاین**: ببینید کدوم کاربرا الان وصلن
- **ریست ترافیک**: وقتی لازمه شمارنده ترافیک کاربر رو ریست کنید

## مدیریت اشتراک

### لینک‌های اشتراک

PasarGuard لینک‌های اشتراک سازگار با کلاینت‌های محبوب رو می‌سازه:

- **V2Ray**: فرمت استاندارد اشتراک V2Ray
- **Clash**: فرمت کانفیگ Clash
- **ClashMeta**: فرمت پیشرفته Clash با قابلیت‌های بیشتر
- **Sing-Box**: کانفیگ json سازگار با Sing-Box
- **Outline**: لینک‌های سازگار با Outline (فقط Shadowsocks رو ساپورت می‌کنه)

### کدهای QR

کدهای QR که به صورت خودکار ساخته می‌شن برای راه‌اندازی آسون کلاینت:

- کدهای QR رو مستقیم با کاربرا به اشتراک بذارید
- راه‌اندازی سریع روی موبایل
- پشتیبانی از همه پروتکل‌ها

## مدیریت نود

### معماری چند نودی

زیرساختتون رو بین چندین سرور توزیع کنید:

- **توزیع بار**: ترافیک رو بین نودها متعادل کنید
- **توزیع جغرافیایی**: نودها رو در مکان‌های مختلف قرار بدید

### مانیتورینگ نود

زیرساخت نود خودتون رو پیگیری کنید:

- **وضعیت سلامت**: اتصال و استاتوس نود رو چک کنید
- **استفاده از منابع**: CPU، حافظه و پهنای باند رو مانیتور کنید
- **آمار ترافیک**: تحلیل ترافیک برای هر نود

## مانیتورینگ سیستم

### آمار ترافیک

مانیتورینگ کامل ترافیک:

- **آمار لحظه‌ای**: نمایش استفاده از ترافیک به صورت زنده
- **داده‌های تاریخی**: تاریخچه و روند ترافیک
- **تحلیل به ازای کاربر**: مصرف هر کاربر به صورت جداگانه
- **نمای کلی سیستم**: مصرف کل پهنای باند

### معیارهای عملکرد

عملکرد سیستم رو مانیتور کنید:

- **مصرف CPU**: میزان استفاده از CPU سرور
- **مصرف حافظه**: استفاده از RAM
- **کانکشن‌های فعال**: تعداد اتصالات همزمان
- **آپتایم**: آمار در دسترس بودن سیستم

## امکانات اتوماسیون

### ربات تلگرام

یکپارچگی با تلگرام برای نوتیفیکیشن و مدیریت:

- **نوتیفیکیشن کاربران**: هشدارهای خودکار برای انقضا و ترافیک
- **کامندهای ادمین**: مدیریت پنل از طریق تلگرام
- **آلرت‌های لحظه‌ای**: نوتیفیکیشن‌های وضعیت سیستم

### ابزارهای CLI

رابط خط فرمان برای اتوماسیون:

```bash
# رابط ترمینال ادمین
pasarguard tui

# مدیریت ادمین
pasarguard cli admin create --sudo
pasarguard cli admin list

# عملیات سیستم
pasarguard status
pasarguard restart
pasarguard update
```

## امکانات امنیتی

### پشتیبانی TLS

اتصالات امن با TLS:

- **گواهی‌های سفارشی**: از گواهی‌های SSL خودتون استفاده کنید
- **کانفیگ SNI**: پشتیبانی از Server Name Indication
