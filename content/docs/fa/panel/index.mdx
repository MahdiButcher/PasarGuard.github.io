# پنل PasarGuard

**راه‌حل یکپارچه GUI مقاوم در برابر سانسور**

پنل PasarGuard یک پلتفرم جامع مدیریت پروکسی است که رابط کاربری وب ساده و بصری برای مدیریت صدها حساب پروکسی ارائه می‌دهد. این پلتفرم با Python و React.js ساخته شده، توسط Xray-core پشتیبانی می‌شود و برای کاربرپسند بودن، غنی از ویژگی و قابل اعتماد بودن طراحی شده است.

## چرا پنل PasarGuard را انتخاب کنیم؟

پنل PasarGuard پیچیدگی مدیریت پروکسی را با ارائه موارد زیر از بین می‌برد:

- **راه‌اندازی آسان**: در عرض چند دقیقه با اسکریپت نصب خودکار شروع کنید
- **رابط بصری**: رابط کاربری وب تمیز و مدرن که به راحتی قابل پیمایش است
- **ویژگی‌های قدرتمند**: ابزارهای مدیریت جامع برای تمام نیازهای پروکسی شما
- **عملکرد بالا**: ساخته شده بر روی فناوری‌های قوی برای عملکرد بهینه
- **توسعه فعال**: به‌روزرسانی‌های منظم و پشتیبانی جامعه

## ویژگی‌های کلیدی

### قابلیت‌های اصلی
- **رابط کاربری وب داخلی** - داشبورد مدرن و پاسخگو
- **بک‌اند REST API** - دسترسی کامل برنامه‌نویسی
- **پشتیبانی چند گره** - توزیع زیرساخت در چندین مکان
- **پشتیبانی پروتکل** - VMess، VLESS، Trojan و Shadowsocks
- **کاربران چند پروتکل** - یک کاربر، چندین پروتکل
- **مدیریت ترافیک** - محدودیت‌های جامع ترافیک و انقضا

### ویژگی‌های پیشرفته
- **لینک‌های اشتراک** - سازگار با کلاینت‌های V2Ray، Clash و ClashMeta
- **تولید کد QR** - لینک‌های اشتراک و کدهای QR خودکار
- **نظارت سیستم** - آمار ترافیک و نظارت در زمان واقعی
- **پیکربندی سفارشی** - گزینه‌های پیکربندی انعطاف‌پذیر Xray
- **امنیت** - پشتیبانی TLS و REALITY
- **خودکارسازی** - ربات تلگرام و ابزارهای CLI یکپارچه
- **چند زبانه** - پشتیبانی از چندین زبان

## گزینه‌های نصب

پنل PasarGuard از چندین بک‌اند پایگاه داده پشتیبانی می‌کند:

- **SQLite** - مناسب برای استقرارهای کوچک و تست
- **MySQL** - ایده‌آل برای محیط‌های تولید
- **MariaDB** - جایگزین منبع باز MySQL
- **PostgreSQL** - ویژگی‌های پیشرفته و مقیاس‌پذیری
