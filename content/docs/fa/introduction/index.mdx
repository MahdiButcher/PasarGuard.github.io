# پنل پاسارگارد (PasarGuard Panel)

**یه راه‌حل کامل و یکپارچه برای دور زدن سانسور - پلتفرم مدیریت پروکسی**

پنل پاسارگارد یه پلتفرم کامل برای مدیریت پروکسی‌ها هست که یه رابط کاربری ساده و راحت بهتون میده تا صدها اکانت پروکسی رو مدیریت کنید. با Python و React.js ساخته شده، از قدرت Xray-core استفاده می‌کنه و طوری طراحی شده که کاربرپسند، پر امکانات و قابل اعتماد باشه.

## چرا پنل پاسارگارد رو انتخاب کنیم؟

پنل پاسارگارد تمام پیچیدگی‌های مدیریت پروکسی رو برطرف می‌کنه و این امکانات رو بهتون میده:

- **نصب آسون**: با اسکریپت نصب خودکار ما، تو چند دقیقه شروع به کار کنید
- **رابط کاربری راحت**: یه رابط وب تمیز و مدرن که راحت باهاش کار کنید
- **امکانات قدرتمند**: ابزارهای کامل مدیریتی برای همه نیازهای پروکسی شما
- **عملکرد عالی**: با تکنولوژی‌های قدرتمند ساخته شده برای بهترین عملکرد
- **توسعه فعال**: آپدیت‌های منظم و پشتیبانی از جامعه کاربران

## امکانات کلیدی

### قابلیت‌های اصلی
- **رابط وب داخلی** - یه داشبورد مدرن و واکنش‌گرا
- **بک‌اند REST API** - دسترسی کامل برنامه‌نویسی
- **پشتیبانی چند نود** - توزیع زیرساخت در چندین لوکیشن
- **پشتیبانی پروتکل** - VMess، VLESS، Trojan و Shadowsocks
- **کاربران چند پروتکلی** - یه کاربر، چندین پروتکل
- **مدیریت ترافیک** - محدودیت‌های کامل ترافیک و زمان انقضا

### امکانات پیشرفته
- **لینک‌های اشتراک** - سازگار با کلاینت‌های V2Ray، Clash، ClashMeta، Sing-box و Outline
- **تولید QR کد** - لینک‌های اشتراک‌گذاری و کیو آر کد به صورت خودکار
- **مانیتورینگ سیستم** - آمار ترافیک و نظارت لحظه‌ای
- **تنظیمات سفارشی** - گزینه‌های پیکربندی انعطاف‌پذیر Xray
- **امنیت** - پشتیبانی از TLS و REALITY
- **اتوماسیون** - ربات تلگرام و ابزارهای CLI یکپارچه
- **چند زبانه** - پشتیبانی از زبان‌های مختلف
