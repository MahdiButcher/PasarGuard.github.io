## معرفی

نود PasarGuard توسط تیم PasarGuard ساخته شده تا یه راه‌حل پایدار، مقیاس‌پذیر و قدرتمند برای مدیریت هسته‌های پروکسی بهتون بده. این نود می‌تونه هم با پنل PasarGuard کار کنه و هم به صورت مستقل استفاده بشه.

### چرا نود PasarGuard؟

نود PasarGuard رو طوری طراحی کردیم که خیلی انعطاف‌پذیر و همه‌کاره باشه:

- **طراحی همه‌کاره**: می‌تونی توی هر پروژه‌ای ازش استفاده کنی، حتی بدون پنل PasarGuard
- **نصب آسون**: با یه اسکریپت ساده و کتابخانه‌های رسمی، نودتو راه‌اندازی کن
- **پشتیبانی از چند هسته**: قراره بعد از تست، از هسته‌های مختلف هم پشتیبانی کنیم
- **معماری مقیاس‌پذیر**: برای کارایی بالا و قابلیت اطمینان ساخته شده

### هسته‌های پشتیبانی شده

| هسته | وضعیت |
| ---- | ------- |
| [Xray-core](https://github.com/XTLS/Xray-core) | ✅ پشتیبانی می‌شه |
| [Sing-box](https://github.com/SagerNet/sing-box) | ⏳ به زودی |
| [V2ray-core](https://github.com/v2fly/v2ray-core) | ⏳ به زودی |

### ویژگی‌های کلیدی

- **پشتیبانی از gRPC و REST API**: پروتکل ارتباطی که دوست داری رو انتخاب کن
- **آمار لحظه‌ای**: مانیتورینگ سیستم، بکند و آمار کاربرا
- **مدیریت کاربر**: همگام‌سازی و مدیریت خودکار کاربرا
- **لاگ زنده**: مشاهده لاگ‌ها به صورت لحظه‌ای
- **کنترل ترافیک**: کنترل و مانیتورینگ ترافیک به ازای هر کاربر
- **پشتیبانی از SSL/TLS**: ساخت و مدیریت خودکار گواهی‌ها

## معماری

نود PasarGuard با این ابزارها ساخته شده است:

- **Go**: بکند با کارایی بالا
- **gRPC & REST API**: پشتیبانی از دو پروتکل
- **Protobuf**: سریال‌سازی کارآمد داده
- **Docker**: استقرار آماده کانتینر
