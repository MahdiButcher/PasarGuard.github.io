# PasarGuard Panel

**统一的反审查图形界面解决方案 - 完整的代理管理平台**

PasarGuard Panel 是一个综合性的代理管理平台，提供简单直观的Web界面来管理数百个代理账户。基于 Python 和 React.js 构建，由 Xray-core 提供支持，设计上注重用户友好性、功能丰富性和可靠性。

## 为什么选择 PasarGuard Panel？

PasarGuard Panel 通过提供以下功能消除了代理管理的复杂性：

- **简单设置**：通过我们的自动化安装脚本，几分钟内即可开始使用
- **直观界面**：简洁、现代的Web界面，易于导航
- **强大功能**：满足您所有代理需求的综合管理工具
- **高性能**：基于强大的技术构建，实现最佳性能
- **积极开发**：定期更新和社区支持

## 主要特性

### 核心功能
- **内置Web界面** - 现代化、响应式的仪表板
- **REST API 后端** - 完整的编程访问
- **多节点支持** - 跨多个位置分布基础设施
- **协议支持** - VMess、VLESS、Trojan 和 Shadowsocks
- **多协议用户** - 单个用户，多个协议
- **流量管理** - 全面的流量和到期限制

### 高级功能
- **订阅链接** - 兼容 V2Ray、Clash、ClashMeta、Sing-box 和 Outline 客户端
- **二维码生成** - 自动生成分享链接和二维码
- **系统监控** - 实时流量统计和监控
- **自定义配置** - 灵活的 Xray 配置选项
- **安全性** - TLS 和 REALITY 支持
- **自动化** - 集成 Telegram Bot 和 CLI 工具
- **多语言** - 支持多种语言
