## 仪表板概述

PasarGuard 面板为管理代理基础设施提供了现代化的 Web 界面。访问仪表板位置：`https://YOUR_DOMAIN:8000/dashboard/`

## 用户管理

### 创建用户

通过直观的界面创建和管理代理用户：

- **多协议**：为单个用户分配多个协议
- **流量限制**：设置每个用户的数据传输限制
- **到期日期**：配置订阅到期日期

### 用户状态

监控和控制用户账户：

- **激活/停用**：即时启用或禁用用户
- **流量使用**：实时监控数据消耗
- **在线状态**：查看当前连接的用户
- **重置流量**：需要时重置用户流量计数器

## 订阅管理

### 订阅链接

PasarGuard 生成与流行客户端兼容的订阅链接：

- **V2Ray**：标准 V2Ray 订阅格式
- **Clash**：Clash 配置格式
- **ClashMeta**：具有附加功能的增强 Clash 格式
- **Sing-Box**：兼容 Sing-Box 的 json 配置
- **Outline**：兼容 Outline 的链接（仅支持 Shadowsocks）

### 二维码

自动生成用于简单客户端配置的二维码：

- 直接与用户共享二维码
- 移动设备上的快速设置
- 支持所有协议

## 节点管理

### 多节点架构

在多个服务器之间分配基础设施：

- **负载分配**：在节点之间平衡流量
- **地理分布**：将节点放置在不同位置

### 节点监控

跟踪节点基础设施：

- **健康状态**：监控节点连接和状态
- **资源使用**：CPU、内存和带宽监控
- **流量统计**：每个节点的流量分析

## 系统监控

### 流量统计

全面的流量监控：

- **实时统计**：实时流量使用显示
- **历史数据**：流量历史和趋势
- **每用户分析**：个人用户消费
- **系统范围概览**：总带宽使用

### 性能指标

监控系统性能：

- **CPU 使用率**：服务器 CPU 利用率
- **内存使用**：RAM 消耗
- **活动连接**：并发连接数
- **正常运行时间**：系统可用性统计

## 自动化功能

### Telegram 机器人

与 Telegram 集成以获取通知和管理：

- **用户通知**：自动到期和流量警报
- **管理员命令**：通过 Telegram 管理面板
- **实时警报**：系统状态通知

### CLI 工具

用于自动化的命令行界面：

```bash
# 管理员终端用户界面
pasarguard tui

# 管理员管理
pasarguard cli admin create --sudo
pasarguard cli admin list

# 系统操作
pasarguard status
pasarguard restart
pasarguard update
```

## 安全功能

### TLS 支持

使用 TLS 的安全连接：

- **自定义证书**：使用您自己的 SSL 证书
- **SNI 配置**：服务器名称指示支持
