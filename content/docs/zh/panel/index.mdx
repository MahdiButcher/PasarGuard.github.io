# PasarGuard 面板

**统一的反审查 GUI 解决方案**

PasarGuard 面板是一个综合的代理管理平台，为管理数百个代理账户提供简单直观的 Web 界面。基于 Python 和 React.js 构建，由 Xray-core 提供支持，旨在用户友好、功能丰富且可靠。

## 为什么选择 PasarGuard 面板？

PasarGuard 面板通过提供以下功能消除了代理管理的复杂性：

- **简单设置**：通过我们的自动安装脚本在几分钟内开始使用
- **直观界面**：清洁、现代的 Web 界面，易于导航
- **强大功能**：满足您所有代理需求的综合管理工具
- **高性能**：基于强大技术构建，实现最佳性能
- **积极开发**：定期更新和社区支持

## 主要功能

### 核心功能
- **内置 Web UI** - 现代、响应式仪表板
- **REST API 后端** - 完整的程序化访问
- **多节点支持** - 跨多个位置分发基础设施
- **协议支持** - VMess、VLESS、Trojan 和 Shadowsocks
- **多协议用户** - 单用户、多协议
- **流量管理** - 全面的流量和过期限制

### 高级功能
- **订阅链接** - 与 V2Ray、Clash 和 ClashMeta 客户端兼容
- **二维码生成** - 自动分享链接和二维码
- **系统监控** - 实时流量统计和监控
- **自定义配置** - 灵活的 Xray 配置选项
- **安全性** - TLS 和 REALITY 支持
- **自动化** - 集成 Telegram Bot 和 CLI 工具
- **多语言** - 支持多种语言

## 安装选项

PasarGuard 面板支持多个数据库后端：

- **SQLite** - 适合小规模部署和测试
- **MySQL** - 适合生产环境
- **MariaDB** - MySQL 的开源替代品
- **PostgreSQL** - 高级功能和可扩展性
