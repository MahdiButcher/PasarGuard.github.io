## 概述

PasarGuard Node 由 PasarGuard 团队开发，为管理代理核心提供稳定、可扩展和高效的解决方案。它设计为与 PasarGuard 面板无缝协作或作为独立解决方案使用。

### 为什么使用 PasarGuard Node？

PasarGuard Node 设计为灵活和通用：

- **通用设计**：可用于任何项目，即使没有 PasarGuard 面板
- **简单集成**：使用官方库通过简单脚本运行节点
- **多核心支持**：计划在测试阶段后扩展支持各种核心
- **可扩展架构**：为性能和可靠性而构建

### 支持的核心

| 核心 | 支持 |
| ---- | ------- |
| [Xray-core](https://github.com/XTLS/Xray-core) | ✅ 已支持 |
| [Sing-box](https://github.com/SagerNet/sing-box) | ⏳ 即将推出 |
| [V2ray-core](https://github.com/v2fly/v2ray-core) | ⏳ 即将推出 |

### 主要功能

- **gRPC 和 REST API 支持**：选择您喜欢的通信协议
- **实时统计**：监控系统、后端和用户统计信息
- **用户管理**：动态用户同步和管理
- **实时日志**：实时流式传输日志
- **流量控制**：每个用户的流量监控和控制
- **SSL/TLS 支持**：内置证书生成和管理

## 架构

PasarGuard 节点使用以下技术构建：

- **Go**：高性能后端
- **gRPC & REST API**：双协议支持
- **Protobuf**：高效的数据序列化
- **Docker**：容器化部署就绪
