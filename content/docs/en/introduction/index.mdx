<h2>PasarGuard, Unified GUI Censorship Resistant Solution</h2>

PasarGuard is a proxy management tool that provides a simple and easy-to-use user interface for managing hundreds of proxy accounts powered by Xray-core and built using Python and Reactjs.

<h3>Why using PasarGuard?</h3>

PasarGuard is user-friendly, feature-rich and reliable. It lets you to create different proxies for your users without any complicated configuration. Using its built-in web UI, you are able to monitor, modify and limit users.

<h3>Features</h3>

<ul>
  <li>Built-in Web UI</li>
  <li>Fully REST API backend</li>
  <li>Multiple Nodes support (for infrastructure distribution & scalability) Docs</li>
  <li>Supports protocols Vmess, VLESS, Trojan and Shadowsocks</li>
  <li>Multi-protocol for a single user</li>
  <li>Multi-user on a single inbound</li>
  <li>Multi-inbound on a single port (fallbacks support)</li>
  <li>Traffic and expiry date limitations</li>
  <li>Periodic traffic limit (e.g. daily, weekly, etc.)</li>
  <li>Subscription link compatible with V2ray (such as V2RayNG, SingBox, Nekoray, etc.), Clash and ClashMeta</li>
  <li>Automated Share link and QRcode generator</li>
  <li>System monitoring and traffic statistics</li>
  <li>Customizable xray configuration</li>
  <li>TLS and REALITY support</li>
  <li>Integrated Telegram Bot</li>
  <li>Integrated Command Line Interface (CLI)</li>
  <li>Multi-language</li>
  <li>Multi-admin support (WIP)</li>
</ul>
