# PasarGuard Panel

**Unified GUI Censorship Resistant Solution - Complete proxy management platform**

PasarGuard Panel is a comprehensive proxy management platform that provides a simple and intuitive web interface for managing hundreds of proxy accounts. Built with Python and React.js, it's powered by Xray-core and designed to be user-friendly, feature-rich, and reliable.

## Why Choose PasarGuard Panel?

PasarGuard Panel eliminates the complexity of proxy management by providing:

- **Easy Setup**: Get started in minutes with our automated installation script
- **Intuitive Interface**: Clean, modern web UI that's easy to navigate
- **Powerful Features**: Comprehensive management tools for all your proxy needs
- **High Performance**: Built on robust technologies for optimal performance
- **Active Development**: Regular updates and community support

## Key Features

### Core Capabilities
- **Built-in Web UI** - Modern, responsive dashboard
- **REST API Backend** - Full programmatic access
- **Multi-Node Support** - Distribute infrastructure across multiple locations
- **Protocol Support** - VMess, VLESS, Trojan, and Shadowsocks
- **Multi-Protocol Users** - Single user, multiple protocols
- **Traffic Management** - Comprehensive traffic and expiry limitations

### Advanced Features
- **Subscription Links** - Compatible with V2Ray, Clash, ClashMeta, Sing-box and Outline clients
- **QR Code Generation** - Automated share links and QR codes
- **System Monitoring** - Real-time traffic statistics and monitoring
- **Custom Configuration** - Flexible Xray configuration options
- **Security** - TLS and REALITY support
- **Automation** - Integrated Telegram Bot and CLI tools
- **Multi-Language** - Support for multiple languages
