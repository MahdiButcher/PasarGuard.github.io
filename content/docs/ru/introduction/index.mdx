<h2>PasarGuard, Унифицированное GUI-решение для обхода цензуры</h2>

PasarGuard — это инструмент управления прокси-серверами, который предоставляет простой и удобный пользовательский интерфейс для управления сотнями прокси-аккаунтов, работающих на базе Xray-core и созданных с использованием Python и Reactjs.

<h3>Почему стоит использовать PasarGuard?</h3>

PasarGuard удобен в использовании, функционален и надежен. Он позволяет создавать различные прокси для ваших пользователей без сложной настройки. Используя встроенный веб-интерфейс, вы можете отслеживать, изменять и ограничивать пользователей.

<h3>Возможности</h3>

<ul>
  <li>Встроенный веб-интерфейс</li>
  <li>Полноценный REST API бэкенд</li>
  <li>Поддержка множественных узлов (для распределения инфраструктуры и масштабируемости)</li>
  <li>Поддержка протоколов Vmess, VLESS, Trojan и Shadowsocks</li>
  <li>Мультипротокол для одного пользователя</li>
  <li>Многопользовательский режим на одном входящем соединении</li>
  <li>Множественные входящие соединения на одном порту (поддержка резервных соединений)</li>
  <li>Ограничения трафика и срока действия</li>
  <li>Периодические ограничения трафика (например, ежедневные, еженедельные и т.д.)</li>
  <li>Ссылка подписки совместима с V2ray (такими как V2RayNG, SingBox, Nekoray и т.д.), Clash и ClashMeta</li>
  <li>Автоматический генератор ссылок для обмена и QR-кодов</li>
  <li>Мониторинг системы и статистика трафика</li>
  <li>Настраиваемая конфигурация xray</li>
  <li>Поддержка TLS и REALITY</li>
  <li>Интегрированный Telegram Bot</li>
  <li>Интегрированный интерфейс командной строки (CLI)</li>
  <li>Многоязычность</li>
  <li>Поддержка нескольких администраторов (в разработке)</li>
</ul>
