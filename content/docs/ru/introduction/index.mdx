# PasarGuard Panel

**Унифицированное GUI-решение для обхода цензуры - Полноценная платформа управления прокси**

PasarGuard Panel — это комплексная платформа для управления прокси-серверами, которая предоставляет простой и интуитивно понятный веб-интерфейс для управления сотнями прокси-аккаунтов. Созданная на Python и React.js, она работает на базе Xray-core и разработана с учетом удобства использования, богатого функционала и надежности.

## Почему стоит выбрать PasarGuard Panel?

PasarGuard Panel устраняет сложности управления прокси, предоставляя:

- **Простая установка**: Начните работу за считанные минуты с помощью нашего автоматизированного скрипта установки
- **Интуитивный интерфейс**: Чистый, современный веб-интерфейс, в котором легко ориентироваться
- **Мощный функционал**: Комплексные инструменты управления для всех ваших прокси-потребностей
- **Высокая производительность**: Построен на надежных технологиях для оптимальной производительности
- **Активная разработка**: Регулярные обновления и поддержка сообщества

## Ключевые возможности

### Основные функции
- **Встроенный веб-интерфейс** - Современная, адаптивная панель управления
- **REST API бэкенд** - Полный программный доступ
- **Поддержка нескольких узлов** - Распределение инфраструктуры по нескольким локациям
- **Поддержка протоколов** - VMess, VLESS, Trojan и Shadowsocks
- **Мультипротокольные пользователи** - Один пользователь, несколько протоколов
- **Управление трафиком** - Комплексные ограничения по трафику и сроку действия

### Расширенные функции
- **Ссылки подписки** - Совместимость с клиентами V2Ray, Clash, ClashMeta, Sing-box и Outline
- **Генерация QR-кодов** - Автоматические ссылки для обмена и QR-коды
- **Мониторинг системы** - Статистика трафика и мониторинг в реальном времени
- **Пользовательская конфигурация** - Гибкие опции конфигурации Xray
- **Безопасность** - Поддержка TLS и REALITY
- **Автоматизация** - Интегрированный Telegram Bot и инструменты CLI
- **Мультиязычность** - Поддержка нескольких языков
