## Обзор панели управления

Панель PasarGuard предоставляет современный веб-интерфейс для управления вашей прокси-инфраструктурой. Получите доступ к панели управления по адресу `https://YOUR_DOMAIN:8000/dashboard/`

## Управление пользователями

### Создание пользователей

Создавайте и управляйте прокси-пользователями через интуитивный интерфейс:

- **Несколько протоколов**: Назначение нескольких протоколов одному пользователю
- **Лимиты трафика**: Установка лимитов передачи данных для каждого пользователя
- **Даты истечения срока**: Настройка дат истечения подписки

### Статус пользователя

Мониторинг и контроль учетных записей пользователей:

- **Активен/Неактивен**: Мгновенное включение или отключение пользователей
- **Использование трафика**: Мониторинг потребления данных в реальном времени
- **Онлайн-статус**: Просмотр пользователей, которые в данный момент подключены
- **Сброс трафика**: Сброс счетчиков трафика пользователей при необходимости

## Управление подписками

### Ссылки на подписку

PasarGuard генерирует ссылки на подписку, совместимые с популярными клиентами:

- **V2Ray**: Стандартный формат подписки V2Ray
- **Clash**: Формат конфигурации Clash
- **ClashMeta**: Расширенный формат Clash с дополнительными функциями
- **Sing-Box**: Совместимая json конфигурация Sing-Box
- **Outline**: Совместимые ссылки Outline (Поддерживает только Shadowsocks)

### QR-коды

Автоматически генерируемые QR-коды для простой настройки клиента:

- Делитесь QR-кодами напрямую с пользователями
- Быстрая настройка на мобильных устройствах
- Поддержка всех протоколов

## Управление нодами

### Архитектура с несколькими нодами

Распределите вашу инфраструктуру по нескольким серверам:

- **Распределение нагрузки**: Балансировка трафика между нодами
- **Географическое распределение**: Размещение нодов в разных местах

### Мониторинг нодов

Отслеживайте вашу инфраструктуру нодов:

- **Состояние здоровья**: Мониторинг подключения и статуса нодов
- **Использование ресурсов**: Мониторинг CPU, памяти и пропускной способности
- **Статистика трафика**: Анализ трафика по нодам

## Мониторинг системы

### Статистика трафика

Комплексный мониторинг трафика:

- **Статистика в реальном времени**: Отображение использования трафика в реальном времени
- **Исторические данные**: История и тенденции трафика
- **Аналитика по пользователям**: Индивидуальное потребление пользователей
- **Общий обзор системы**: Общее использование пропускной способности

### Метрики производительности

Мониторинг производительности системы:

- **Использование CPU**: Утилизация CPU сервера
- **Использование памяти**: Потребление RAM
- **Активные соединения**: Количество одновременных соединений
- **Время работы**: Статистика доступности системы

## Функции автоматизации

### Telegram бот

Интеграция с Telegram для уведомлений и управления:

- **Уведомления пользователей**: Автоматические предупреждения об истечении срока и трафике
- **Команды администратора**: Управление панелью через Telegram
- **Уведомления в реальном времени**: Уведомления о состоянии системы

### Инструменты CLI

Интерфейс командной строки для автоматизации:

```bash
# Терминальный интерфейс администратора
pasarguard tui

# Управление администраторами
pasarguard cli admin create --sudo
pasarguard cli admin list

# Системные операции
pasarguard status
pasarguard restart
pasarguard update
```

## Функции безопасности

### Поддержка TLS

Безопасные соединения с TLS:

- **Пользовательские сертификаты**: Используйте свои собственные SSL-сертификаты
- **Конфигурация SNI**: Поддержка Server Name Indication
