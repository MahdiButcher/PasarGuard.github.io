# Панель PasarGuard

**Унифицированное GUI решение, устойчивое к цензуре**

Панель PasarGuard - это комплексная платформа управления прокси, которая предоставляет простой и интуитивно понятный веб-интерфейс для управления сотнями прокси-аккаунтов. Построенная на Python и React.js, она поддерживается Xray-core и разработана для удобства использования, богатства функций и надежности.

## Почему выбрать панель PasarGuard?

Панель PasarGuard устраняет сложность управления прокси, предоставляя:

- **Простая настройка**: Начните за несколько минут с помощью нашего автоматического скрипта установки
- **Интуитивный интерфейс**: Чистый, современный веб-интерфейс, по которому легко ориентироваться
- **Мощные функции**: Комплексные инструменты управления для всех ваших потребностей в прокси
- **Высокая производительность**: Построена на надежных технологиях для оптимальной производительности
- **Активная разработка**: Регулярные обновления и поддержка сообщества

## Ключевые функции

### Основные возможности
- **Встроенный веб-интерфейс** - Современная, отзывчивая панель управления
- **REST API Backend** - Полный программный доступ
- **Поддержка нескольких узлов** - Распределение инфраструктуры по нескольким локациям
- **Поддержка протоколов** - VMess, VLESS, Trojan и Shadowsocks
- **Многопротокольные пользователи** - Один пользователь, несколько протоколов
- **Управление трафиком** - Комплексные ограничения трафика и срока действия

### Продвинутые функции
- **Ссылки подписки** - Совместимость с клиентами V2Ray, Clash и ClashMeta
- **Генерация QR-кодов** - Автоматические ссылки для обмена и QR-коды
- **Мониторинг системы** - Статистика трафика и мониторинг в реальном времени
- **Пользовательская конфигурация** - Гибкие опции конфигурации Xray
- **Безопасность** - Поддержка TLS и REALITY
- **Автоматизация** - Встроенный Telegram Bot и CLI инструменты
- **Многоязычность** - Поддержка нескольких языков

## Варианты установки

Панель PasarGuard поддерживает несколько бэкендов баз данных:

- **SQLite** - Идеально для небольших развертываний и тестирования
- **MySQL** - Подходит для производственных сред
- **MariaDB** - Открытая альтернатива MySQL
- **PostgreSQL** - Продвинутые функции и масштабируемость
